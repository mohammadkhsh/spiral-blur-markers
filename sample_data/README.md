# Sample Data

This repository includes a small sample pack from the final sharp-marker validation split.

Contents:
- `sharp/images/`: sample rendered scenes
- `sharp/labels/`: YOLO-format spiral labels for Stage 1
- `sharp/annotations/`: full scene metadata with robot-level and spiral-level ground truth

The sample pack is intended for format inspection, quick tests, and figure reproduction. It is not the full training dataset.
